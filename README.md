# Документация по проекту

## Описание проекта
Проект ревлизован на основании open-source утилиты Masscan и языка программирования Python.

## Задачи проекта
Задачей проекта является:
- Выполнять многопоточное сканирование адресов.
- Определять открытые порты
- Проводить анализ баннеров
- Проверять историю обнаружение портов, на выявление был ли этот порт открыт ранее
- Уведомление о событиях

## Стек используемых технологий
- masscan 1.3.2
- Python3.12
- API telegram


## Установка компонентов проекта

```bash
sudo apt update
sudo apt install -y masscan

pip install -r requirements.txt
```

## Запуск проекта

Для выполнения полного цикла сканирования, обогащения и уведомления запустите
главный скрипт:

```bash
python -m app.scripts.main
```

Скрипт последовательно выполнит:

1. **Masscan** – многопоточное сканирование согласно `app/config.json`.
2. **Nmap** – получение баннеров и определение сервисов для открытых портов.
3. **Анализ** – сравнение текущего скана с предыдущим и формирование отчёта о
   изменениях.
4. **Telegram** – отправка сообщения в чат, если обнаружены новые или изменённые
   хосты/порты (см. раздел *notifications* в конфигурации).

Все результаты сохраняются в директории `app/scan_history/`. Файл
`latest_scan.json` содержит последние скан‑данные, а `changes.json` – diff с
предыдущим сканом.
